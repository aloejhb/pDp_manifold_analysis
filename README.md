# Code for generating figures for the zebrafish pDp olfactory neural manifold analysis

This repository contains the code used to generate all figures for the paper:

Hu, B., Temiz, N. Z., Chou, C.-N., Rupprecht, P., Meissner-Bernard, C., Titze, B., Chung, S., & Friedrich, R. W. (2024).  
*Representational learning by optimization of neural manifolds in an olfactory memory network*.  
bioRxiv. https://doi.org/10.1101/2024.11.17.623906

## Requirements

To run the code, please install the required package:

```bash
pip install git+https://github.com/aloejhb/catrace
```
